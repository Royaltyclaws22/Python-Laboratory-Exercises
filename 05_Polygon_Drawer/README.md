# 🎨 Polygon Drawer

A simple Tkinter-based drawing application that allows users to create and fill polygons by selecting points on a canvas. Users can choose colors from a palette or customize them using the built-in color picker.

## ⚙️ Features

- Create polygons by selecting points on a canvas.
- Connect selected points with lines to preview the polygon shape.
- Fill completed polygons with the selected color.
- Choose colors from a predefined palette.
- Customize colors using the color picker.
- Simple and lightweight graphical interface.


## 💻 Installation & Running

1. Clone the repository:
```bash
git clone https://github.com/Royaltyclaws22/Python-Laboratory-Exercises.git
```
2. Navigate to the project folder:
```bash
cd Python-Laboratory-Exercises/05_Polygon_Drawer
```
3. Run the program:
```bash
python Polygon_Drawer.py
```


## 💡 Usage

- `Left click` on the canvas to add points.
- Points are automatically connected with lines.
- `Double left click` to complete and fill the polygon.
- Click a color in the palette to select the drawing color.
- `Right click` on a color to customize it using the color picker.
- To create a new polygon, continue selecting points on the canvas after the previous polygon has been completed.
- Close the application window when you have finished drawing.
