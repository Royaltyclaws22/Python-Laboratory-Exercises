# 🎵 Closest Note

A simple Python program that finds and displays the closest musical note (name, octave, and frequency) for a given input frequency.


## 🔎 Preview
<img width="1280" height="680" alt="Closest_Note" src="https://github.com/user-attachments/assets/6d198aaa-a2ee-462b-a810-5a72114e16a9" />


## ⚙️ Features

- Supports frequencies from 20Hz to 22000Hz.
- Finds the closest musical note based on frequency distance.
- Selects the lower frequency when two notes are equally close.
- Displays the note name, octave, and frequency.
- Handles invalid user input.


## 💻 Installation & Running

1. Clone the repository:
```bash
git clone https://github.com/Royaltyclaws22/Python-Laboratory-Exercises.git
```
2. Navigate to the project folder:
```bash
cd Python-Laboratory-Exercises/03_Closest_Note
```
3. Run the program:
```bash
python Closest_Note.py
```


## 💡 Usage

Enter an integer frequency when prompted.

**Example:**

```text
Enter frequency (0 to exit): 444
Closest note: La4 (440Hz)
```

Enter `0` to exit the program.
