# △ Triangle Analyzer 

Triangle Analyzer is a simple Python program that accepts the lengths of three sides, verifies whether they form a valid triangle, and calculates its basic geometric properties in order to determine its type.


## ⚙️ Features

- Reads three side lengths from user input
- Sorts the sides in descending order
- Checks the triangle inequality
- Calculates the perimeter
- Calculates the area using Heron's formula
- Identifies whether the triangle is:
  - Equilateral
  - Isosceles
- Classifies the triangle by its angles:
  - Acute
  - Right
  - Obtuse


## 💻 Installation & Running

1. Clone the repository:
```bash
git clone https://github.com/Royaltyclaws22/Python-Laboratory-Exercises.git
```
2. Navigate to the project folder:
```bash
cd Python-Laboratory-Exercises/01_Triangle_Analyzer
```
3. Run the program:
```bash
python Triangle_Analyzer.py
```


## 💡 Usage

Enter the three side lengths separated by commas when prompted.

**Example:**

```text
INPUT: 3,4,5
```

The program will display:

- The sorted side lengths
- The perimeter
- The area
- The triangle type (equilateral or isosceles, if applicable)
- The angle classification (acute, right, or obtuse)
